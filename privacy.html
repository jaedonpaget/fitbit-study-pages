Privacy Policy

Effective date: [2026-01-20]

1. Summary

This App accesses Fitbit heart-rate data via the Fitbit Web API after you authorize access through Fitbit’s consent screen. The App is intended for the App owner’s personal/academic use.

2. Data we collect

  • Fitbit heart-rate intraday data (timestamped heart-rate values).
  • OAuth tokens provided by Fitbit (access token and refresh token) to access the API.
  • Basic technical logs needed to run the script (e.g., timestamps of requests/errors).

3. How we use data

  • To download heart-rate data from Fitbit and save it to local files (e.g., CSV/JSON) on your computer.
  • To maintain access to the Fitbit API by refreshing OAuth tokens when required.

4. Where data is stored

Data is stored locally on the App owner’s computer. The App does not intentionally upload your Fitbit data to any external server.

5. Sharing

The App does not sell Fitbit data and does not share Fitbit data with third parties, except that Fitbit processes API requests as part of providing the Fitbit platform.

6. Retention

Data is kept only as long as needed for the personal/academic project and can be deleted at any time by removing the locally saved files.

7. Security

Reasonable measures are used to protect locally stored files (for example, keeping the computer account private and not publishing token files). However, no method of storage is 100% secure.

8. Your choices

  • You can revoke the App’s access at any time from your Fitbit account settings.
  • You can delete the locally stored files at any time.
